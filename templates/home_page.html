RFID Cashless System

Working on a system to make monetary exchanges cashless in a closed community using existing unique RFID. The system integrates card reader, raspberry Pi (Running Raspian) and an application using flask. Working in a team and individually developed GUI interface using Tkinter and front end for flask application to run on Pi. The project is mentored by  Prof. Kumar Padmanabh.

i. Number of comparisions w.r.t. a randomized input array.
ii. Running Time w.r.t. a randomized input array.
A shell script, GNU plot along with the algos coded in C were used.
